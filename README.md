## Overview

This repository publishes the skill files for the Command Compass desktop workflow in a GitHub-friendly bilingual format.

- Chinese package: `AI Bookmark & Desktop Dock Assistant (ZH)`
- English package: `Bookmark & Desktop Dock AI Assistant`
- Package type: OpenClaw-compatible skill package
- Desktop app: [Command Compass official download](https://www.wboke.com/zh/download)

## Why This Repo Exists

- Helps discovery platforms index the skill files from a public repository
- Gives users a clean place to review the skill files before using them
